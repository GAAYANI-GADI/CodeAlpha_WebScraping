# CodeAlpha Web Scraping Project

## Objective
To extract structured data from a public website using Python web scraping techniques.

## Website Used
https://quotes.toscrape.com/

## Data Collected
- Quote text
- Author name

## Tools & Libraries
- Python
- Requests
- BeautifulSoup
- Pandas

## Project Structure
- data/: Contains the scraped CSV dataset
- notebooks/: Jupyter notebook explaining the scraping process
- src/: Python script for web scraping

## Output
A CSV file (`quotes_data.csv`) is generated and stored in the data folder.

## Conclusion
This project demonstrates basic web scraping skills and data extraction using Python.
